# Yukii Calculator Bot

A simple Telegram calculator bot inspired by @Yukii_calculator_bot.

## Features
- Basic arithmetic operations: +, -, *, /
- Parentheses support: ( )
- Exponentiation: ^
- Thousand separator formatting
- Inline buttons for Copy and Delete

## Deployment
This bot is designed to be deployed on Render.
1. Push this code to a GitHub repository.
2. Create a new "Background Worker" on Render.
3. Connect your GitHub repository.
4. Set the Environment Variable `TELEGRAM_TOKEN`.
